КОНТЕКСТ для Windows. 
Приложение к Руководству пользователя. 
Настройка КОНТЕКСТа на текстовый процессор. 
 
В поставку КОНТЕКСТа включены макрокоманды, позволяющие 
использовать его как функцию текстовых процессоров Microsoft Word 
версий 2.0 и 6.0, а также Lotus AmiPro версии 3.0.  
Программа настройки КОНТЕКСТа создаcт в меню Вашего текстового 
процессора позицию, служащую для вызова КОНТЕКСТа, так что 
КОНТЕКСТ после настройки превращается во встроенную функцию 
текстового процессора, подобную функции проверки орфографии или 
расстановки переносов. Это позволяет вызывать КОНТЕКСТ через меню 
текстового процессора, а также даёт возможность выбрать 
дополнительные "быстрые клавиши" и ввести кнопку для быстрого вызова 
словаря, используя для этого обычные средства настройки текстового 
процессора. 
 
Итак, если Вы хотите настроить КОНТЕКСТ на Ваш текстовый процессор, 
нужно выполнить следующие действия: 
 
Войдите в группу КОНТЕКСТа, созданную при установке КОНТЕКСТа на 
жесткий диск. Первые три значка в группе  (с изображением попугая, 
компьютера и блокнота) служат соответственно для запуска словаря, 
добавления или удаления словарей и для вывода на экран текстового файла 
README с  последними сведениями о данной версии КОНТЕКСТа. 
Остальные значки служат для настройки КОНТЕКСТа на различные 
текстовые процессоры. Если среди них есть значок, соответствующий 
Вашему текстовому процессору, Вам достаточно будет дважды щёлкнуть 
по нему мышью. Дальнейшее развитие диалога будет зависеть от того, 
какой текстовый процессор Вы выбрали.
 
Ниже описывается настройка КОНТЕКСТа на MS Word и AmiPro.
 
1. Настройка КОНТЕКСТа на текстовый процессор Microsoft Word версий 
2.0 и 6.0
 
Программа настройки КОНТЕКСТа встраивает вызов КОНТЕКСТа в 
меню Сервис (Tools) текстового процессора MS Word.
 
1.1. Подготовка к настройке
Прежде чем вызвать настройку, убедитесь, что в данный момент MS Word 
не является активным (то есть, что его нет в списке открытых окон 
Windows). В противном случае по окончании встройки КОНТЕКСТа у Вас 
будут проблемы при сохранении Всеобщего шаблона (Global Template) при 
выходе из Word. 
Если всё-таки Ms Word был загружен ранее, закройте его  окно  клавишами 
ALT + F4.
 
1.2. Настройка КОНТЕКСТа 
Теперь для запуска Программы настройки войдите в программную группу 
КОНТЕКСТа и дважды щёлкните мышью по значку, соответствующему 
MS Word. Программа настройки вызовет MS Word, через некоторое время 
MS Word загрузится и на его экране Вы увидите текст инструкции 
Программы настройки. Пожалуйста, внимательно прочтите эту 
инструкцию. На самом деле, Вам просто нужно дважды щёлкнуть мышью 
по кнопке с надписью "Установить КОНТЕКСТ / Install CONTEXT", 
которая находится прямо в тексте инструкции. 
Вслед за этим MS Word исполнит макрокоманду, "привязанную" к этой 
кнопке и в меню Сервис (Tools) появится позиция "Context", служащая для 
вызова КОНТЕКСТа.
 
1.3. Настройка MS Word ("быстрые клавиши" и кнопка быстрого вызова) 
Для ускорения вызова КОНТЕКСТа Вы можете привязать к данной 
позиции меню "быстрые клавиши", а также ввести соответствующую 
кнопку в Строку инструментов (Toolbar).  
Позиция меню "Context" описывается в MS Word макрокомандой с именем 
Context, поэтому Вам нужно выбрать "быстрые клавиши" и кнопку именно 
для этой макрокоманды. Как это сделать, Вы можете посмотреть в 
Руководстве к MS Word.
 
1.4. Сохранение настройки 
Теперь Вы можете выйти из MS Word. Перед выходом MS Word спросит 
Вас, нужно ли сохранить Всеобщий шаблон (Global Template). Чтобы 
позиция "Context", привязанные к ней "быстрые клавиши" и кнопка в 
Строке инструментов не исчезли при следующем вызове MS Word, Вам 
нужно ответить "да" на этот вопрос.
 
2. Настройка КОНТЕКСТа на текстовый процессор Lotus AmiPro 3.0
 
С помощью Программы настройки КОНТЕКСТа Вы можете ввести вызов 
КОНТЕКСТа в меню  Инструменты (Tools) текстового процессора AmiPro.
 
2.1. Подготовка к настройке 
Прежде чем начать настройку КОНТЕКСТа на AmiPro, убедитесь, что в 
данный момент AmiPro не активен (то есть, что окна AmiPro нет в списке 
открытых окон Windows). Если всё-таки AmiPro был загружен ранее, 
закройте его окно клавишами ALT + F4.
 
2.2. Настройка КОНТЕКСТа 
Теперь для запуска Программы настройки войдите в программную группу 
КОНТЕКСТа и дважды щёлкните мышью по значку, соответствующему 
AmiPro. Программа настройки вызовет AmiPro и на экране Вы увидите 
текст  инструкции Программы настройки. 
Смысл её сводится к тому, что для настройки КОНТЕКСТа Вам нужно 
дважды щёлкнуть мышью по кнопке с надписью "Установить КОНТЕКСТ 
/ Install CONTEXT", которая находится прямо в тексте инструкции. 
Вслед за нажатием  кнопки AmiPro исполнит макрокоманду, 
"привязанную" к этой кнопке и в меню  Инструменты (Tools) появится 
позиция "Контекст" ("Context"), служащая для вызова КОНТЕКСТа.
 
2.3. Настройка AmiPro ("быстрые клавиши" и кнопка быстрого вызова) 
Для ускорения вызова КОНТЕКСТа Вы можете связать с данной позицией 
меню "быстрые клавиши", а также добавить соответствующую 
Пиктограмму (SmartIcon).  
Позиция меню "Контекст" ("Context") описывается в файле макрокоманды 
для вызова КОНТЕКСТа под именем CONTEXT.SMM, находящемся на 
жестком диске в каталоге КОНТЕКСТа. Именно к этому файлу 
макрокоманды Вам нужно привязать "быстрые клавиши" и Пиктограмму 
(SmartIcon). Как это делается, подробно описано в Вашем Руководстве к 
AmiPro.
