 Обозначения:
  HV  - mode: view file as hex/ascii dump
  AV  - mode: view file as assembler instruction with codes
  TV  - mode: view file as text
  ╬   - новое.
  ■   - есть и работает.
  +   - исправлено или улучшено.
  √   - работает, но иногда глючит.
  -   - отсутствует, но скоро будет ;)
  ~   - авторские ремарки. ;)

      те, которые вышли на широкие просторы с моего компьютера.
