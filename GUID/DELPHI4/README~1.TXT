Вступение

 На прилагаемом компакт-диске содержатся файлы всех демонстрационных 
проектов, описываемых в книге. Скопируйте эти файлы с каталогов компакт-
диска в новый каталог на жестком диске и откройте скопированный проект 
с помощью команды Delphi File|Open или File|Open Project. В нужном 
каталоге откройте файл с расширением .dpr -- как правило, в проекте он 
один. Если поиск файла с расширением .dpr вызывает трудности, нужно 
открыть диалоговое окно Open и убедиться, что в поле Files of Type 
(тип файла) выбран тип файлов Delphi (расширение *.pas, *.dpr). Однако 
учтите, что не все каталоги содержат файлы проектов.

Как использовать листинги

 Все приводимые в книге листинги можно найти в указанных каталогах 
прилагаемого компакт-диска. Причем, среди этих файлов нет 
заархивированных. Поэтому для их использования достаточно скопировать 
нужный подкаталог каталога Source на жесткий диск. Если есть место, можно 
скопировать весь каталог Source на диск C:\ или D:\.

Некоторые читатели просят, чтобы я организовал  приложения в 
соответствии с главами книги. Во многих случаях такая организация 
учебных программ весьма полезна. Но в данной книге в разных главах 
встречаются ссылки на одно и то же приложение. По этой причине все 
приложения распределены по каталогам, имеющим такое же название, как и 
соответствующие им проекты. Например, файлы проекта SysColor находятся 
в каталоге Source\SysColor.  Хотя большинство каталогов используется 
для хранения в них отдельных проектов Delphi, в некоторых, как , 
например, Misc, находятся только файлы с расширением .pas или другие. 
Для их просмотра можно воспользоваться командой File|Open.

Компилирование и запуск программ

 Все предлагаемые программы уже откомпилированы. Для их выполнения 
можно воспользоваться услугами Windows Explorer и запустить 
программный файл с расширением .exe. Большинство программ можно 
запускать непосредственно с компакт-диска. Однако существует целый ряд 
программ, написанных специально для работы с жестким диском. Вначале их 
нужно скопировать на жесткий диск, и уже только после этого запускать.  

Для компиляции программы сначала нужно скопировать на жесткий диск 
файлы из соответствующего каталога. Если попытаться выполнить 
компиляцию прямо на компакт-диске, Delphi выдаст сообщение об ошибке 
при попытке записи в .exe-файл. Если и после копирования каталога на 
жесткий диск при компиляции возникают проблемы, то, скорее всего, файлы 
помечены как Read-Only, т.е. только для чтения. В этом случае 
обратитесь к следующему разделу.

Файлы для чтения

 Скопированные с компакт-диска файлы могут иметь флаги, разрешающие 
только чтение этих файлов. Это может вызвать определенные проблемы при 
попытке записи в программные файлы .exe и .dcu. Если при компиляции 
программы появляется сообщение, что файл невозможно использовать или что 
он предназначен только для чтения, нужно выполнить следующие действия.

 1. Скопируйте каталог Source с компакт-диска на жесткий диск в каталог 
    C:\, D:\ или любой другой.

 2. Откройте окно MS-DOS.

 3. В командную строку введите C: или D: (может быть другая литера)  
    для выхода из каталога Source.

 4. Для перехода в каталог Source введите команду CD \Source.

 5. Командой ATTRIB -r *.* /S сбросьте флаги Read-Only файлов.

 6. Если при выполнении получено сообщение об ошибке "Sharing 
    violation", нужно выйти из Delphi и повторить п. 4 еще раз.
